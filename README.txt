PASSO A PASSO PARA PUBLICAR NO GITHUB PAGES:
1. Entre no GitHub e clique em New repository
2. Dê um nome, por exemplo: stylofarma-itatiaia
3. Marque Public
4. Clique em Create repository
5. Clique em Add file > Upload files
6. Envie os arquivos index.html, styles.css e script.js
7. Clique em Commit changes
8. Vá em Settings
9. No menu lateral, clique em Pages
10. Em Source, selecione Deploy from a branch
11. Em Branch, escolha main e pasta /root
12. Clique em Save
13. Aguarde a publicação
14. O link final ficará parecido com:
   https://SEU-USUARIO.github.io/stylofarma-itatiaia/
